Porque usar tags semânticas?

Tags semânticas dão significado ao conteúdo, facilitam a leitura do código, melhoram a acessibilidade e ajudam os motores de busca a entender a estrutura do site. Evitar excesso de <div> deixa o código mais limpo e organizado.

Tags HTML comuns

Algumas das tags semânticas que podes usar para estruturar teu site:

  • <header>: Cabeçalho da página ou de secção
  • <nav>: Menú de navegação
  • <main>: Conteúdo principal da página
  • <section>: Agrupamento de conteúdo relacionado
  • <article>: Conteúdo independente como um Post de um Blog
  • <aside>: Conteúdo complementar ou lateral
  • <footer>: Rodapé da página ou de secção
  • <h1> a <h6>: Títulos hierárquicos
  • <p>: Define um parágrafo
  • <ul> / <li>: Cria listas ordenadas / define itens de lista
  • <a>: Cria links
  • <img>: Insere imagens na página
  • <strong>: Dá ênfase ao texto
  • <span>: Container genérico em linha
